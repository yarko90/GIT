Тестовое задание

При запуске main.py:
1) создаются пустые таблицы для зданий, этажей, офисов, комнат, датчиков в базе данных PostgreSQL(модуль Create_tables.py)
2) в таблицу buildings записываются 5 зданий(building0-building4),
   для каждого здания в таблицу floors записывается два этажа (floor0, floor0 - floor4,floor4),
   для каждого этажа два офиса и так далее.(Для создания тестовых зданий: модуль Tester, для записи в БД:класс Adder в модуле DB_Processor.py)
3) В бесконечном цикле для получившихся 80 детекторов случайным образом добавляются случайные замеры температуры, 
   раз в десять секунд для каждого детектора замеры усредняются .(Для создания тестовых замеров: модуль Tester, для записи замеров в БД:класс Updater в модуле DB_Processor.py)
   В результате в базе данных детекторов каждую минуту обновляются первые 6 столбцов времени(hour1-hour6)
4) Для вывода информации о суточной температуре для объекта(здание, этаж, офис, комната, детектор) используется класс Getter из модуля DB_Processor.py. 
   Для примера, выводятся показания для каждого из тестовых зданий.
